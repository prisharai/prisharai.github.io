# pr15ha portfolio

A cinematic multi-page portfolio for pr15ha built with React 18, TypeScript,
Vite, Tailwind CSS 3, React Router, and Lucide React.

## Development

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually `http://127.0.0.1:5173/`.

## Validation

```bash
npm run lint
npm run build
```
